# الاسم الرقم الوظيفي المسمى الإدارة الخبرة الفريق تاريخ الإرسال الإجابات
🧠 Agent 1 — محلل الأتمتة
يحلل إجابات الموظفين ويحدد أفضل فرص الأتمتة بترتيب الأولوية
كيف يعمل؟ يقرأ كل إجابات الموظفين ويخرج قائمة مرتبة بأفضل 10 فرص للأتمتة مع تقدير الوقت الموفَّر لكل فرصة.
⏳ Agent 2 — ملخص الوقت الضائع
يكتب تقريراً تنفيذياً للإدارة العليا عن الوقت المستهلك
كيف يعمل؟ يجمع بيانات توزيع الوقت ويكتب تقريراً تنفيذياً جاهزاً للإدارة العليا مع التوصيات.
🎯 Agent 3 — المستشار الاستراتيجي
يبني خطة استراتيجية لـ 10-30 AI Agent حسب بيانات الشركة
كيف يعمل؟ يستوعب كامل بيانات الشركة ويبني خطة AI Agents مرتبة بالأثر، سهولة التنفيذ، التكلفة، والجدول الزمني.
⚙️ إعدادات النظام
احصل على مفتاحك مجاناً من openrouter.ai/keys
لماذا OpenRouter؟ يتيح استخدام Claude Sonnet وGPT-4o وGemini بمفتاح واحد بتكلفة منخفضة جداً.
